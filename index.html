StarClusterPlugins by jtriley

Collection of user-contributed plugins for StarCluster

Collection of user-contributed plugins for StarCluster. If you're interested in contributing your own plugin, please join the StarCluster mailing list and then fork the project on github to begin contributing the code. Have a look at ....(TODO) for coding guidelines and instructions.

Dependencies

This package relies on the StarCluster package which can be installed from pypi using easy_install or downloaded installed using the standard 'python setup.py intall' method.

Install

To install the user-contributed plugins package for StarCluster, download and extract the package to your local hard drive and then run: $ python setup.py install
